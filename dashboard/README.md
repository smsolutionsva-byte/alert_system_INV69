# Dashboard (Vite + React + Tailwind + Tremor)

## Install

```powershell
cd "c:\Users\Red\Downloads\INV69 project\dashboard"
npm install
```

## Configure Firebase

```powershell
copy .env.example .env
```

Set all `VITE_FIREBASE_*` values in `.env`.

## Run

```powershell
npm run dev
```

## Build

```powershell
npm run build
npm run preview
```

## Optional: shadcn/ui CLI

```powershell
npx shadcn@latest init
```
